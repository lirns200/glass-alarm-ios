# Настройка проекта для ИИ (Quick Setup)

1. **Корень**: Убедись, что ты находишься в папке `GlassAlarm`.
2. **Конфиг**: Проверь наличие `builder.json` в корне.
3. **Зависимости**: Установи Python 3.10+ и зависимости если нужно.
4. **Ресурсы**: Убедись, что `GlassAlarm/Resources/crystal.wav` существует.
5. **Пуш**: Сборка идет через GitHub, проверь доступ к репозиторию.
6. **Билд**: Для создания уникального IPA всегда используй `AI_Dev_Kit/ai_build.py`.
7. **Проверка**: Все `.ipa` файлы падают в папку `dist/`.
8. **Логи**: Если билд упал, смотри логи в `ipa_build_workspace/`.
9. **Секрет**: Для тестов мини-игры — 5 быстрых тапов по часам.
